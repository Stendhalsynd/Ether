# SPEC.md

## 1. 게임 개요
- 게임명(가칭): `에테르의 눈`
- 장르: 모바일 우선 샌드박스 로그라이크 RPG
- 핵심 콘셉트: 무한 탐험(네피아 런) + 고정 메인 플롯 + 절차적 미시 서사 결합
- 정체성: “에테르는 위협이자 항체”라는 이중 진실을 중심으로 한 선택 기반 서사형 성장 게임

## 2. 목표와 KPI
- 목표(런칭 3개월):
  - 핵심 튜토리얼 완주율 85%+
  - 7일 리텐션 30% 이상
  - 시즌당 최소 1회 복귀 유도율 45% 이상
- 장기 목표: 시즌 스토리로 운영 콘텐츠와 런 복귀 동기를 동시 성장

## 3. 핵심 게임 루프
### 3.1 세션 루프(런)
1. 사전 준비(동료/장비/부품 선택)
2. 네피아 진입(탐사/전투/이벤트/증거 획득)
3. 위험-보상 트레이드오프 의사결정
4. 귀환 및 연대기(Chronicle) 자동 생성
5. 시즌·증거 보드 반영으로 다음 런 설계

### 3.2 메타 루프
- 에테르/메세라 축척 수치 변화
- 증거 조각 해석 및 진실 판정
- 길드·시즌 이벤트 참여
- 시즌별 패치/스토리 적용

## 4. 시스템 요구사항

### 4.1 메인 플롯(Authorial)
- 3부 구조(프롤로그/초기/중기/엔드) 고정
- 4개 핵심 NPC(로미아스/라네이레/사이모어/세비리스)의 정보 독점/거짓/거래/배신 축으로 분기
- 시즌별 메인 플롯 진척 노출 방식이 점층적으로 열림

### 4.2 절차적 미시 서사(Procedural)
- 네피아 런마다 사건 모듈이 가중치 기반 생성
- 사건 분류:
  - 오염의 증거
  - 항체의 대가
  - 진실의 거래
  - 선택의 기록
- 사건 결과가 `증거 보드`와 `선택 가중치`에 반영

### 4.3 캐릭터 및 관계
- 로미아스: 생존 가이드 + 초반 진입 제약 장치
- 라네이레: 비용 기반 진실 해설/데이터 번역
- 사이모어: 문명 생존 동원 + 영웅화 역설
- 세비리스: 정보 유통 비대칭 및 반전 허브

### 4.4 네피아 모듈
- 던전 층수/환경/몬스터/미션 조합형 생성
- 3개 마석:
  - 풍석: 계절/네피아 변동 예측 권한(예고 강화)
  - 혈석: 감염 지연 권한(거래형 대가 부여)
  - 성석: 정화 이벤트 강제/완화 권한
- 고유 보스는 메인 플롯 분기 트리거 아이템 또는 증거 지급

### 4.5 UI/서사 지원
- 연대기 탭(Chronicle): 런 종료 후 선택/손실/획득 자동 요약
- 증거 보드(Proof Board): 증거 카테고리 수집·연결·조합
- 풍석 기반 시즌 예고(Season Forecast): 다음 런/시즌 동기 부여 힌트 제공

### 4.6 시즌 운영
- 기본 주기: 39일 시즌
- 주차 1: 변종 폭풍 이벤트 + 개시 퀘스트
- 주차 2~4: 네피아 패턴 변형, 신규 증거 세트
- 주차 5~6: 길드 협동 과제 + 시즌 보스
- 운영 원칙: 보상 압박보다 세계 이해도 보상(정보·해석) 우선

## 5. 기술/비기능 요구
- 대상 플랫폼: Android/iOS
- 모바일 45fps 목표(안정 동작 우선)
- 첫 접속 후 10초 내 핵심 상태 진입
- 플레이 로그 30일 저장, 주요 선택/증거/시즌 상태 영구 보존

## 6. 데이터 타입(최소 인터페이스)
- `RunContext { runId, seed, party, loadouts, riskMod, objectiveFlags }`
- `RunEvent { eventId, source, tags, rarity, rewards, consequences }`
- `LoreFact { id, sourceType, certainty, factionsImpacted, worldEffectId }`
- `TruthBundle { id, shardIds[], interpretedBy, timestamp, unlockedAct }`
- `SeasonState { seasonId, phase, activeMutations, narrativeTheme, bossConfig }`

## 7. 수치/밸런스
- 에테르 노출: 단기 보존력 상승 / 장기 변형 리스크 증가
- 메세라 수치: 즉시성 낮음 / 누적 페널티 큼
- 선택 기록 가중치: 반복 선택 패턴 기반 사건 확률 조정
- 리스크 가시화: 사운드·텍스트·환경색 변화로 즉시 인지

## 8. 리스크
- 무한 RNG 피로: 증거 및 연대기 결합이 약하면 급증
- 시즌 피로: 시간 압박형 과제 남용 시 이탈 가능
- 내러티브 파편화: 대사-이벤트 연동 미스매치 위험

## 9. 검증 기준
- 20분 런에서 기본 루프 정상 완료율
- 핵심 NPC 3개 이상 구간별 일관성 있는 정보 공개
- 연속 3런 선택 기록이 다음 런 이벤트 가중치 반영되는지 확인
- 시즌 전환 후 첫 2런 내 보드 반응 동작
- 엔드 분기 최소 4개 노출
